### 👨‍💻 Sobre
Graduando em Análise e Desenvolvimento de Sistemas pela UNICID, concentro meus estudos e projetos no desenvolvimento Backend e na Análise de Dados. Minha trajetória é pautada pela resolução lógica de problemas e pela busca constante em transformar requisitos complexos em soluções tecnológicas eficientes e escaláveis. Com uma base sólida em engenharia de prompt e ferramentas no-code, busco integrar a inteligência de dados ao desenvolvimento de software para otimizar processos e gerar valor real por meio da tecnologia."
 
* **Habilidades Técnicas:** Python, Java, SQL, HTML, CSS, IA Generativa.
* **Habilidades Socioemocionais:** Trabalho em equipe, comunicação assertiva e pensamento crítico, muito paciente, procuro resolver os problemas na melhor situação possível.
